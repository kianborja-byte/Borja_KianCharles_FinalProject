# Past Electricity Bills Tracker ⚡

A Command-Line Interface (CLI) application built in Python to help users record, track, and analyze their historical electricity consumption and costs.

# Project Overview
This tool allows users to maintain a digital log of their electricity meter readings. It goes beyond simple recording by automatically calculating costs based on a **tiered pricing model** and providing deep insights through trend analysis and statistical summaries. It was developed as a final project for Python programming.

# Features
-   **Tiered Pricing Calculation:** Automatically calculates costs using a two-tier system (lower rate for the first 100 kWh, higher rate for excess).
-  **Advanced Sorting:** View your billing history chronologically, by highest/lowest cost, or by energy consumption (kWh).
-  **Trend Analysis:** Compares current usage against previous months with percentage change indicators (📈/📉).
-  **Statistical Summary:** Generates a report showing total paid, average bill cost, average kWh used, and price extremes.
-  **Tagging System:** Organize bills with custom labels (e.g., "summer", "aircon", "vacation") for targeted searching.
-   **Tier Breakdown:** A transparency feature that shows exactly how many kWh were charged at each price level for every bill.
